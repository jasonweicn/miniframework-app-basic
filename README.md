MiniFramework Basic Application Template
====================

这是一套基于 MiniFramework 的基础应用模板，开发者可以通过这个模板快速开始构建自己的 Web 应用。

通过 Composer 安装
====================

php composer.phar create-project --prefer-dist --stability=dev jasonweicn/miniframework-app-basic
